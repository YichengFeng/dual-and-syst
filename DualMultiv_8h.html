Detailed Description

Multi-variable dual number for first-order automatic differentiation.

Tracks partial derivatives per variable ID via a sorted std::map<int, double>. Supports all common arithmetic and math functions, plus weighted-average merging and correlated-systematic uncertainty computation via a static correlation matrix.
