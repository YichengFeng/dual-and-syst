Detailed Description

Free-function operators and utilities for ROOT TGraphErrors and TGraphAsymmErrors.

Provides arithmetic operators (+ - * /), conversion functions, centrality rebinning, uncertainty propagation helpers (Barlow-style difference and ratio), event-plane resolution calculation, and various graph manipulation utilities.

Author
Yicheng Feng Email: fengy.nosp@m.ich@.nosp@m.outlo.nosp@m.ok.c.nosp@m.om

Improvements over old_code version:

  • Removed using namespace std (namespace pollution in header)
  • Removed const from operator return-by-value (enables move semantics)
  • Replaced VLAs (double xe[n]) with std::vector<double>
  • Fixed operator%: no longer mutates input graph x-values in place
  • Replaced 0.0/0.0 with std::nan("")
  • Added explicit includes: TF1, TGraphAsymmErrors, iomanip, sstream
  • Consistent formatting throughout
