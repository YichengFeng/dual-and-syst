Detailed Description

A graph with systematic uncertainty variations (ROOT-independent).

Holds a default DualGraph and a map of systematic variation DualGraphs keyed by integer vids. Provides systematic evaluation via evaluate_systematics(), LaTeX table generation via to_latex(), and full arithmetic operator support with vid-aligned variation propagation.

See also
TSystGraph (ROOT-aware counterpart), SystMergeMode, SystVariation, SystResult

SystResult SystGraph::evaluate_systematics ( const std::vector< int > &  vids) const
inline

Evaluate systematics for a specified subset of vids.

Evaluate systematics for specified vids.

Parameters
vidslist of variation IDs to evaluate
Returns
a SystResult (skips non-existent vids).

For each variation, computes the raw difference \( v - d \), applies the Barlow correction if configured, and accumulates quadrature-summed totals. One-sided systematics contribute to either the upper or lower total based on the sign of the systematic.
