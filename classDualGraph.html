Detailed Description

A graph of DualPoint objects supporting uncertainty propagation.

Provides arithmetic operators (+ - * /), math functions (pow, sqrt, exp, log, sin, cos, tan, atan, atan2), weighted/unweighted bin averaging, and bin selection. This is the ROOT-independent base class; for ROOT I/O use TDualGraph.
